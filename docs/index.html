Andesaurux

Andesaurux is a fully free OS that is aimed at easiest package management and freedom.

1. Andesaurux is an OS. 2. It's fully free. 3. It is protected by copyleft BSD license. 4. There is a pkgm, so you can download apps. 5. It is in active development. 6. You can contribute to it.
